# 🌸 Iris Dataset PCA Analysis

This project demonstrates **Principal Component Analysis (PCA)** on the Iris dataset using Python and Scikit-learn.

## ✨ Features
- Load the Iris dataset
- Standardize the data using `StandardScaler`
- Reduce dimensions from **4 features to 2 principal components** using PCA
- Visualize the transformed data with a PCA scatter plot
- Explore feature relationships using a pair plot
- Display the explained variance retained by each principal component

## 🛠️ Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## 📌 Output
- 2D PCA scatter plot of the Iris dataset
- Pair plot of the original features
- Explained variance ratio of the principal components
